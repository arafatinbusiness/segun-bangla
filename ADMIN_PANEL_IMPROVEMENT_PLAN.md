# অ্যাডমিন প্যানেল উন্নয়ন পরিকল্পনা

## 📋 বর্তমান সমস্যা

অ্যাডমিন প্যানেলে নতুন নিবন্ধ তৈরি/সম্পাদনার সময় ক্যাটাগরি ও সাবক্যাটাগরি (বিশেষ করে জেলা) নির্বাচন করতে **অনেক স্ক্রল করতে হয়**। বর্তমানে এগুলো চেকবক্স গ্রিড আকারে দেখানো হয়, যা ৬০+ জেলার জন্য খুবই অসুবিধাজনক।

---

## 🎯 সমাধান: ৪-কলামের কম্প্যাক্ট প্যানেল

বর্তমান ফর্মের ক্যাটাগরি/সাবক্যাটাগরি অংশ প্রতিস্থাপন করে একটি **৪-কলামের স্ক্রলেবল প্যানেল** তৈরি করা হবে।

### কলাম ১ — স্পেশাল (Special Articles)
- **শিরোনাম:** স্পেশাল
- **বিষয়বস্তু:** ১০টি চেকবক্স — প্রধান নিবন্ধ + বিশেষ নিবন্ধ ১ থেকে ৯
- **ফিচার:** মাউস স্ক্রল করে যেকোনো একটিতে টিক দেওয়া যাবে
- **বর্তমান ফিল্ড:** `isLead` (প্রধান) ও `isSpecial` (বিশেষ) — এগুলো UI-তে ১০টি অপশন আকারে দেখানো হবে

### কলাম ২ — ক্যাটাগরি (Categories)
- **শিরোনাম:** ক্যাটাগরি
- **বিষয়বস্তু:** "সর্বশেষ" বাদে সব ক্যাটাগরি (সব দেখুন এর ক্যাটাগরিগুলো সহ)
- **ফিচার:** 
  - মাউস স্ক্রল করে এক বা একাধিক নির্বাচন
  - নির্বাচিত ক্যাটাগরির ভিত্তিতে নিচে সাব-ক্যাটাগরি দেখাবে
  - **জেলার ক্ষেত্রে শুধু প্রশাসনিক বিভাগ** (রংপুর, রাজশাহী, ঢাকা, ইত্যাদি) দেখাবে
- **ডেটা সোর্স:** বর্তমান `getAllCategories()` + `getSubcategoriesByCategory()`

### কলাম ৩ — জেলা (District)
- **শিরোনাম:** জেলা
- **বিষয়বস্তু:** কলাম ২-এ নির্বাচিত বিভাগের জেলাগুলো
- **ফিচার:** মাউস স্ক্রল করে একটি জেলা নির্বাচন
- **ডেটা সোর্স:** বর্তমান সাবক্যাটাগরি — `parentId` দিয়ে ফিল্টার

### কলাম ৪ — উপজেলা (Sub-district)
- **শিরোনাম:** উপজেলা
- **বিষয়বস্তু:** কলাম ৩-এ নির্বাচিত জেলার উপজেলাগুলো
- **ফিচার:** মাউস স্ক্রল করে এক বা একাধিক উপজেলা নির্বাচন
- **ডেটা সোর্স:** বর্তমান সাবক্যাটাগরি — `parentId` দিয়ে ফিল্টার

## 🗄️ ডেটা মডেল — কোনো পরিবর্তন নেই!

```typescript
// lib/types.ts — বর্তমান, অপরিবর্তিত থাকবে
export interface Subcategory {
  id: string
  name: string
  slug: string
  categoryId: string
  parentId?: string | null  // ← এই ফিল্ড ইতিমধ্যেই আছে!
  order: number
}
```

**কীভাবে কাজ করবে:**
- **বিভাগ (Division):** `parentId: null` — যেমন: রংপুর, রাজশাহী, ঢাকা
- **জেলা (District):** `parentId: divisionId` — যেমন: রংপুর বিভাগের অধীনে কুড়িগ্রাম, লালমনিরহাট
- **উপজেলা (Upazila):** `parentId: districtId` — যেমন: কুড়িগ্রাম জেলার অধীনে উলিপুর, নাগেশ্বরী

এগুলো সবই বর্তমান `Subcategory` টাইপ ও `parentId` ফিল্ড ব্যবহার করবে। **ডাটাবেজে কোনো পরিবর্তন লাগবে না।**
