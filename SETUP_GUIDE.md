# সেগুন বাংলা - সেটআপ গাইড

## প্রজেক্ট সম্পূর্ণ হয়েছে! 🎉

আপনার বাংলাদেশের শীর্ষস্থানীয় সংবাদ পোর্টাল এখন সম্পূর্ণভাবে তৈরি এবং চালু করার জন্য প্রস্তুত।

## তৈরি কিছু

### 1. ফ্রন্টএন্ড কম্পোনেন্ট
- **Header**: নেভিগেশন এবং ক্যাটাগরি সাইডবার সহ রেসপন্সিভ হেডার
- **Footer**: সম্পূর্ণ ফুটার লিঙ্ক এবং কপিরাইট তথ্য সহ
- **ArticleCard**: তিনটি ভ্যারিয়েন্ট (ডিফল্ট, ফিচার্ড, স্মল) সহ পুনর্ব্যবহারযোগ্য আর্টিকেল কার্ড

### 2. পৃষ্ঠা
- **হোমপেজ** (`/`): জটিল 65/35 স্প্লিট লেআউট, ফিচার্ড নিবন্ধ, এবং ক্যাটাগরি পংক্তি
- **আর্টিকেল পৃষ্ঠা** (`/article/[slug]`): পূর্ণ আর্টিকেল বিষয়বস্তু, সম্পর্কিত নিবন্ধ, SEO মেটাডেটা
- **ক্যাটাগরি পৃষ্ঠা** (`/category/[slug]`): ফিল্টার্ড নিবন্ধ গ্রিড এবং সাব-ক্যাটাগরি ট্যাব
- **অনুসন্ধান পৃষ্ঠা** (`/search?q=...`): গতিশীল অনুসন্ধান ফলাফল
- **404 পৃষ্ঠা**: কাস্টম "পৃষ্ঠা পাওয়া যায়নি" ডিজাইন

### 3. ডেটা সেবা
- **Articles Service**: শীর্ষস্থানীয়, বৈশিষ্ট্যযুক্ত, সর্বশেষ নিবন্ধ এবং অনুসন্ধানের জন্য কার্যকর
- **Categories Service**: ক্যাটাগরি এবং সাব-ক্যাটাগরি পরিচালনা

### 4. Firebase ইন্টিগ্রেশন
- সম্পূর্ণ Firebase সেটআপ (Firestore, Storage, Auth রেডি)
- টাইপ-নিরাপদ সেবা স্তর
- সার্ভার-সাইড ডেটা ফেচিং

### 5. SEO অপটিমাইজেশন
- মেটাডেটা প্রতিটি পৃষ্ঠার জন্য
- JSON-LD স্ট্রাকচার্ড ডেটা
- OG ট্যাগ সোশ্যাল মিডিয়া শেয়ারিংয়ের জন্য
- ক্যানোনিকাল URL

## দ্রুত শুরু করতে

### ধাপ 1: Firebase প্রজেক্ট তৈরি করুন

1. https://console.firebase.google.com এ যান
2. নতুন প্রজেক্ট তৈরি করুন
3. Firestore Database সক্ষম করুন
4. Storage সক্ষম করুন
5. আপনার ওয়েব অ্যাপের জন্য Firebase প্রজেক্ট সেটিংস থেকে শংসাপত্র কপি করুন

### ধাপ 2: পরিবেশ ভেরিয়েবল সেট করুন

```bash
# আপনার প্রজেক্টে .env.local ফাইল তৈরি করুন এবং যোগ করুন:
NEXT_PUBLIC_FIREBASE_API_KEY=আপনার_API_কী
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=আপনার_অথ_ডোমেইন
NEXT_PUBLIC_FIREBASE_PROJECT_ID=আপনার_প্রজেক্ট_আইডি
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=আপনার_স্টোরেজ_বাকেট
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=আপনার_সেন্ডার_আইডি
NEXT_PUBLIC_FIREBASE_APP_ID=আপনার_অ্যাপ_আইডি
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

### ধাপ 3: Firestore নিরাপত্তা নিয়ম সেট করুন

Firebase কনসোলে নিয়ম ট্যাবে যান এবং আপডেট করুন:

```firestore rules
rules_version = '3';
service cloud.firestore {
  match /databases/{database}/documents {
    // সর্বজনীন পড়ুন অনুমতি
    match /{document=**} {
      allow read: if true;
      allow write: if false; // শুধুমাত্র অ্যাডমিন অ্যাপ থেকে লেখুন
    }
  }
}
```

### ধাপ 4: নমুনা ডেটা দিয়ে বীজ বপন করুন

```bash
pnpm seed-db
```

এটি নিম্নলিখিত নমুনা ডেটা তৈরি করবে:
- 5 ক্যাটাগরি (দেশীয় সংবাদ, আন্তর্জাতিক, ক্রীড়া, বিনোদন, প্রযুক্তি)
- প্রতি ক্যাটাগরিতে 3টি সাব-ক্যাটাগরি
- 3টি নমুনা নিবন্ধ

### ধাপ 5: উন্নয়ন সার্ভার চালু করুন

```bash
pnpm dev
```

ব্রাউজারে খুলুন: http://localhost:3000

## ট্রাবলশুটিং

### Firebase সংযোগ সমস্যা?
- সমস্ত পরিবেশ ভেরিয়েবল সঠিক আছে তা নিশ্চিত করুন
- Firebase কনসোলে Firestore সক্ষম আছে তা যাচাই করুন
- নিরাপত্তা নিয়ম সঠিকভাবে সেট আছে তা পরীক্ষা করুন

### ইমেজ লোড হচ্ছে না?
- Firebase Storage নিয়ম পড়ার অনুমতি দেয় তা নিশ্চিত করুন
- ইমেজ URL বৈধ আছে তা যাচাই করুন

### স্থাপনা সমস্যা?
- Vercel ড্যাশবোর্ডে পরিবেশ ভেরিয়েবল যোগ করুন
- বিল্ড লগ ত্রুটিগুলির জন্য পরীক্ষা করুন
- Firebase প্রকল্প ID সঠিক আছে তা নিশ্চিত করুন
